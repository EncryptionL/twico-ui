# Twico UI — documentation site

The documentation website for Twico UI, built with **Vite + React** and **dogfooding Twico UI
itself** — every button, badge, and the dark-mode toggle are real library components. Live at
**https://encryptionl.github.io/twico-ui/**.

## Develop

```bash
cd site
npm install
npm run dev      # local dev server
npm run build    # production build -> site/dist
npm run preview  # preview the production build
```

The site imports the library straight from source via a Vite alias (`twico-ui` → `../src/index.ts`),
so it always reflects the latest components and the example `import` statements match what consumers
write.

## How it's structured

- `src/pages/` — Home, Installation, Theming, Dark mode, Accessibility, and the component index/detail pages.
- `src/components/` — docs chrome (Layout, Sidebar, CodeBlock, LiveExample, PropsTable, ThemeToggle).
- `src/data/components.js` — **auto-generated** component reference (summaries + props tables). Regenerated by the `twico-docs-extract` workflow.
- `src/demos/<Name>Demo.jsx` — one self-contained live demo per component, rendered in an error boundary.

## Deploy

Pushing to `main` (or running the **Deploy docs site** workflow manually) builds the site and
publishes it to GitHub Pages. Enable it once under **Settings → Pages → Source = GitHub Actions**.
