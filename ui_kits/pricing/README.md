# Pricing — UI kit

Marketing pricing page built from Twico UI primitives.

- `index.html` — three plan tiers (Free / Pro / Team) with a monthly⇄yearly **Switch**, a featured
  plan highlighted with a primary border + "Most popular" **Badge**, feature checklists, CTA **Button**s,
  and an **Accordion** FAQ. Prices update live when the billing toggle flips. Layout + copy only.
