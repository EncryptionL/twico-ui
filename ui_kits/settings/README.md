# Settings — UI kit

Account settings screen built from Twico UI primitives.

- `index.html` — a centered settings page with **Tabs** (Profile · Notifications · Billing · Security).
  Profile uses `Avatar` + `Input`/`Select`/`Textarea`; Notifications uses `Switch` rows; Billing shows
  plan + payment `Badge`/`Button` rows; Security has password `Input`s, a 2FA `Switch`, and a danger-zone
  delete `Button`. Layout + copy only — all controls are real components.
