Compact status label or count. Six tones, three fill styles.

```jsx
import { Badge } from "./Badge";

<Badge tone="success" dot>Active</Badge>
<Badge tone="warning" variant="solid">Pending</Badge>
<Badge tone="neutral" variant="outline">Draft</Badge>
```

Tones: primary/success/warning/danger/info/neutral. Variants: soft/solid/outline. `dot` adds a leading dot.
